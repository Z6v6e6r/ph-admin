Параметры генерации

Сначала задайте сценарий турнира, затем нажмите «Сгенерировать сетку». Вопросики рядом с параметрами объясняют, за что каждая настройка отвечает и в какую сторону двигает алгоритм.

Эта страница вызывает POST /api/tournaments/generate-schedule на том же домене. То есть вы тестируете реальный сервис, а не демонстрационный мок.

Симуляция рейтинга

Введите итог одного матча по очкам и нажмите «Рассчитать рейтинг», чтобы увидеть `before / delta / after` по точной формуле игрового модуля.

Для генерации сетки используется обычный турнирный `rating`, а для перерасчёта после сыгранных матчей используется `game rating` по шкале игрового модуля. Это помогает отдельно тестировать качество сетки и отдельно проверять механику изменения рейтинга. В Americano здесь моделируется один матч до заданной суммы очков, например до `21` или `24`.

Сначала сгенерируйте сетку, затем заполните итоговый счёт по каждому матчу и запустите симуляцию рейтинга.
