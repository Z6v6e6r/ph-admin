Public tournaments dev page

Эта страница нужна, чтобы локально и на стенде гонять тот же flow, что пойдёт в Tilda: каталог турниров, real LK auth через /lk_new, join-flow, выбор уровня, обычную запись и waitlist. Справа остаётся живой JSON, чтобы было проще отлаживать контракт backend-а.

Что проверяем

Каталог из /api/tournaments/public/list
Join-flow из /api/tournaments/public/:slug/join?format=json
Шаг AUTH_REQUIRED и popup для padlhub.ru/lk_new
Подбор уровня, обычную запись и добавление в лист ожидания
